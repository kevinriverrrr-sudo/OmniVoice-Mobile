# Contributing to OmniVoice Mobile

Спасибо за интерес к проекту! Вот как вы можете внести вклад.

## 🤝 Как помочь

### Bug Reports

1. Откройте [Issue](https://github.com/omnivoice-mobile/omnivoice-mobile/issues)
2. Укажите:
   - Модель устройства и SoC
   - Версию Android
   - Версию Termux (`termux-info`)
   - RAM устройства (`cat /proc/meminfo`)
   - Полный лог ошибки
   - Шаги для воспроизведения

### Feature Requests

1. Проверьте что такой запрос ещё не существует
2. Откройте Issue с тегом `[FEATURE]`
3. Опишите:
   - Что вы хотите добавить
   - Зачем это нужно
   - Предлагаемый подход

### Pull Requests

1. Fork репозиторий
2. Создайте ветку: `git checkout -b feature/my-feature`
3. Внесите изменения
4. Добавьте тесты если применимо
5. Commit: `git commit -m "Add my feature"`
6. Push: `git push origin feature/my-feature`
7. Откройте Pull Request

### Требования к коду

- Python 3.10+
- PEP 8 стиль
- Типизация (type hints)
- Документация для публичных функций
- Тесты для новых функций

## 🏷️ Теги Issues

| Тег | Описание |
|-----|----------|
| `[BUG]` | Ошибка / баг |
| `[FEATURE]` | Запрос новой функции |
| `[PERF]` | Улучшение производительности |
| `[DEVICE]` | Проблема с конкретным устройством |
| `[DOC]` | Улучшение документации |

## 📋 Кодекс поведения

- Будьте уважительны к другим контрибьюторам
- Принимайте конструктивную критику
- Фокусируйтесь на том, что лучше для проекта
- Пишите на русском или английском

Спасибо за ваш вклад! ❤️
