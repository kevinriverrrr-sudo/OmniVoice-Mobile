USAGE
 
omnivoice <command> [options]
 
COMMANDS
 
generate Generate speech from text
clone Clone a voice from reference audio
list-voices List available TTS voices
list-langs List supported languages
info Show device and model information
update Update to the latest version
uninstall Remove OmniVoice Mobile
 
GENERATE OPTIONS
 
-t, --text <string> Text to synthesize
-f, --file <path> Text file to synthesize
-l, --lang <code> Language code (default: en)
-v, --voice <name> Voice name (default: en-US-AriaNeural)
-c, --clone <dir> Use cloned voice profile
-o, --output <path> Output file path (default: output.wav)
--speed <float> Speech speed 0.5–2.0 (default: 1.0)
--pitch <float> Pitch shift -10 to +10 (default: 0)
 
CLONE OPTIONS
 
-s, --source <audio> Reference audio file (5–30s)
-n, --name <string> Name for the voice profile
-o, --output <dir> Output profile directory
 
GLOBAL OPTIONS
 
-h, --help Show this help message
-V, --version Print version information
--verbose Enable verbose logging
--quiet Suppress non-error output
 
EXAMPLES
 
# Basic text-to-speech
omnivoice generate -t "Hello World" -l en -o hello.wav
 
# Clone a voice and generate
omnivoice clone -s ref.wav -n myvoice
omnivoice generate -t "Hi there" -c ~/.ov-voices/myvoice
 
~ $
